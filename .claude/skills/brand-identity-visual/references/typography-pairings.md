# Typography pairings — referencias rápidas

Pairings probados (headline + body), todos disponibles en Google Fonts a menos que se diga lo contrario.

## Por personalidad

### Moderno minimalista (SaaS, tech moderno, DTC clean)
- **Inter** (headline) + **Inter** (body) — el más seguro, multi-weight.
- **Geist** (headline) + **Geist** (body) — Vercel's, geométrica clara.
- **General Sans** (headline) + **General Sans** (body) — Fontshare, premium feel.

### Editorial / premium
- **Playfair Display** (headline) + **Inter** (body) — serif elegante + sans neutro.
- **Tiempos Headline** (headline, pago) + **Söhne** (body, pago) — niveles superiores.
- **Cormorant Garamond** (headline) + **Lato** (body) — más femenino refinado.

### Wellness / orgánico
- **Fraunces** (headline) + **Manrope** (body) — serif suave + sans calmado.
- **DM Serif Display** (headline) + **DM Sans** (body) — misma familia, coherente.

### Bold / Gen-Z / DTC con personalidad
- **Clash Display** (headline, Fontshare) + **Inter** (body).
- **Boldonse** (headline, Fontshare) + **General Sans** (body).
- **Migra** (headline, Pangram Pangram) + **Inter** (body).

### Tech serio / fintech
- **Inter Display** (headline) + **Inter** (body).
- **Suisse Int'l** (headline, pago) + **Suisse Int'l** (body).

### Luxe minimalista (fashion premium)
- **Canela** (headline, pago) + **Söhne** (body).
- **EB Garamond** (headline) + **Inter** (body) — versión gratuita.

### Playful / kids / fun
- **Inter Display** + **Inter** — versátil.
- **Recoleta** (headline) + **Inter** (body) — serif curvilínea cálida.

## Sistema de tamaños (default)

| Nivel | Tamaño px | Tamaño pt | Weight |
|---|---|---|---|
| H1 | 56-72 | 42-54 | Bold |
| H2 | 36-48 | 27-36 | Semibold |
| H3 | 24-32 | 18-24 | Medium |
| Body | 16-18 | 12-14 | Regular |
| Caption | 12-13 | 9-10 | Regular |
| Button | 14-16 | 10.5-12 | Semibold |

Ajusta según display vs body. Para web responsive, considera fluid type (clamp).

## Reglas

1. **Headline serif + body sans** = clásico editorial.
2. **Headline sans bold + body sans regular** = moderno SaaS / DTC.
3. **NUNCA** dos serifs ni dos sans iguales — necesitas contraste o usa una sola familia con weights distintos.
4. **Una sola fuente bien usada > dos mal pareadas.**
5. **Verifica peso de carga**: cada fuente extra son 30-100kb. Para web considera variable fonts.
6. **License check**: Google Fonts es seguro. Adobe Fonts requiere Creative Cloud. Pangram Pangram tiene gratuitas y de pago. Lineto / Klim / Commercial Type son de pago serio.
