# Color psychology — referencias rápidas

Úsalo solo para anclar tus propuestas, no para imponer reglas rígidas.

## Significados generales

| Color | Asocia con | Riesgo | Categorías donde funciona bien |
|---|---|---|---|
| Rojo | Energía, urgencia, pasión, hambre | Agresividad, peligro | F&B, sale, deporte, entretenimiento |
| Naranja | Optimismo, calidez, accesibilidad, juventud | Barato | Comida casual, tech accesible, kids |
| Amarillo | Felicidad, atención, optimismo | Cobardía, ojos cansados como base | F&B, kids, advertencia |
| Verde | Naturaleza, salud, dinero, crecimiento | Genérico "eco" | Wellness, finanzas, sustentabilidad |
| Azul | Confianza, profesionalismo, calma | Frío, corporativo aburrido | Finanzas, tech, salud, B2B SaaS |
| Morado | Lujo, creatividad, espiritualidad | Muy femenino estereotipado | Belleza, premium, creativo |
| Rosa | Femineidad moderna, suavidad, joven | Cliché de género | DTC mujer, belleza, lifestyle |
| Café | Tierra, artesanal, premium-rústico | Sucio si mal usado | Café, chocolate, artesanía, premium clásico |
| Negro | Lujo, autoridad, modernidad | Demasiado serio | Premium, fashion, tech minimalista |
| Blanco | Pureza, minimalismo, espacio | Vacío si abusas | Premium, healthtech, minimalismo |
| Beige / nude | Calma, neutralidad sofisticada | Aburrido | Wellness, slow fashion, hogar |

## Paletas comunes por industria (puntos de partida)

- **DTC wellness**: terrosos + 1 acento vivo. Ej: oliva profundo + crema + naranja quemado.
- **DTC fashion contemporáneo**: monocromo + 1 color signature. Ej: negro/blanco + naranja fuerte.
- **SaaS B2B serio**: azul/morado profundo + neutros + acento brillante.
- **Foodtech**: rojo o naranja + crema + verde acento.
- **Fintech moderno**: negro/blanco + 1 verde o azul ácido como signature.
- **Beauty premium**: rosado nude + dorado + negro.
- **Tech accesible (Gen-Z)**: lavanda + amarillo limón + blanco roto.

## Reglas prácticas

1. **5-6 colores máximo**: primario, secundario, acento, neutro claro, neutro oscuro, semántico (opcional).
2. **Contraste accesible**: el texto sobre el primario debe pasar WCAG AA (4.5:1 mínimo).
3. **Distingue color de marca vs color de UI**: tu primario puede ser muy saturado, pero los neutros tienen que ser cómodos en pantallas grandes.
4. **Considera daltonismo**: si tu acento es rojo y necesitas comunicar "atención", asegúrate de que también haya señalización por forma/icono.
5. **Cliente, no industria**: si el cliente ideal es ejecutivo formal, evita amarillo neón aunque el competidor lo use.

## Anti-clichés

- 🚫 Solo azul + blanco para "tech serio" (todo el mercado se ve igual).
- 🚫 Verde para "eco" sin diferenciación adicional.
- 🚫 Rosa nude para "femenino aspiracional" sin razón estratégica.
- 🚫 5 tonos de gris sin un color signature (la marca desaparece).
