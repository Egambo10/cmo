# TikTok formats — diferencias clave vs Reels

TikTok y Reels son primos, no gemelos. Aquí las diferencias prácticas.

## Diferencias importantes

| Aspecto | TikTok | Reels |
|---|---|---|
| Duración óptima | 15-30s para reach, hasta 60s para retención | 9-15s para reach, 30s OK |
| Audio | Trending sounds = oro. Algoritmo lo premia agresivamente. | Trending sounds importan pero menos. |
| Tono | Más casual, "raw", UGC-feel | Más editorial OK, "made for IG" se nota |
| Texto en pantalla | Estilo TikTok (font default white-on-shadow) funciona | Texto custom branded funciona mejor |
| CTA | "Sígueme para más" funciona ahí | Cliché en IG, evítalo |
| Algoritmo | Te muestra a no-followers agresivamente | Más sesgado a tu audiencia ya seguidora |

## Plantillas TikTok-first

### Storytime
- "Voy a contarles algo que me pasó la semana pasada…" → narrativa de 30-45s con cortes
- Lo importante: voz auténtica, no producción exagerada

### Day in the life (DITL)
- Muy popular en TikTok, menos en Reels
- Plano-secuencia o mini-montaje de momentos
- Música ambiental + voiceover

### Greenscreen reaction
- TikTok premia este formato
- Te paras frente a un screenshot/imagen y reaccionas
- En Reels existe pero menos común

### Duet / Stitch
- Solo TikTok. Reacciona a o continúa un video viral.

## Reglas TikTok

1. **Usa trending sounds**: incluso si es solo de fondo bajo. El algoritmo escucha.
2. **Hook visual + auditivo en el primer segundo**: ambos canales.
3. **Subtitle obligatorio**: TikTok lo genera auto, pero si haces caption custom, mejor.
4. **3 jump cuts mínimo en primer 5s**: mantiene atención.
5. **CTA puede ser más directo**: "Sígueme", "Comenta X", "Dale like si…"

## Prompts para Higgsfield

Para TikTok, prioriza Veo 3.1 sobre Kling. Veo da look "casero genuino" que TikTok premia. Kling es más cinematográfico (mejor para Reels editoriales).

Vertical 9:16, 8 second clip. Authentic UGC style — feels like recorded with iPhone, slight handheld shake, not over-produced. Realistic, slightly imperfect — this should feel like a TikTok, not a commercial. No on-screen text.
