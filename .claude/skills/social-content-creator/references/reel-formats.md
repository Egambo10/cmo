# Reel formats — patrones que funcionan

Para Reels y TikToks de 8-30 segundos. Estos son los esqueletos. El contenido específico siempre se construye sobre la voz y el posicionamiento de la marca.

## Estructura universal

```
[0-2s]  HOOK — frena el scroll
[2-5s]  CONTEXT — quién/qué (rápido)
[5-25s] PAYOFF — el contenido prometido
[25-30s] CTA — qué hacer ahora
```

## Plantillas por objetivo

### Awareness / hook viral

- **Pattern**: "POV: te acaba de pasar X" → demuestra → punchline
- **Pattern**: "No vas a creer lo que…" → revelación visual fuerte
- **Pattern**: "3 cosas que nadie te dice de X" → lista rápida con jump cuts

### Educacional

- **Pattern**: "Cómo hacer X en 30 segundos" → 3-5 pasos rápidos
- **Pattern**: "Antes/Después" → resultado primero, proceso después
- **Pattern**: "Mito vs realidad" → mito en pantalla, voz desmiente

### Promo de producto

- **Pattern**: "Yo antes / yo después con X" → transformación con producto
- **Pattern**: "¿Estás cansado de X? Te presento Y" → problem-solution clásico
- **Pattern**: "Día en la vida usando X" → integración orgánica

### UGC-style testimonial

- **Pattern**: "Llevaba 3 meses buscando X y por fin…" → talking head, casero
- **Pattern**: "Honestamente no creía pero…" → escepticismo → conversión
- **Pattern**: "Si tienes X problema, NECESITAS este Y" → recomendación directa

## Reglas

1. **Caption + on-screen text**: el video se ve sin audio el 80% del tiempo. Texto en pantalla es obligatorio.
2. **Vertical 9:16**: siempre. 16:9 es para YouTube, no para Reels/TikTok.
3. **Hook fuerte**: si el primer segundo no engancha, pierdes al 70%.
4. **Loop**: si puedes hacer que el final empate con el inicio, el algoritmo te premia con vueltas.
5. **Música**: TikTok premia trending sounds; Instagram también pero menos agresivo.
6. **Subtítulos**: siempre, con la tipografía de marca si es legible a 9:16.

## Prompts para Higgsfield (Veo 3.1 / Kling 3.0)

Para un reel, tu prompt debe describir **cada escena** con detalle visual + duración. Veo y Kling generan ~8s por shot — encadénalos o usa multi-shot.

Ejemplo de scene prompt:

```
Vertical 9:16, 8 second clip. Young woman, mid-20s, latin features, sitting at a sunlit kitchen table with her laptop open, smiling and gesturing to the screen as she scrolls. Warm natural light, shallow depth of field, cinematic but UGC-leaning. Brand palette accents visible on the laptop screen (subtle, #FF5733 and #FFD700 only). No on-screen text — that'll be added in post. Camera locked, no shake. Realistic, not over-stylized.
```
